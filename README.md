# Banking Transaction Management System

## Overview
Engineered a secure backend microservices architecture for a banking transaction platform. This system implements robust JWT authentication and data encryption protocols to safeguard sensitive financial data, alongside optimized MongoDB schemas for high-throughput transaction logging and rapid history retrieval.

## Tech Stack
- **Language:** Java
- **Framework:** Spring Boot
- **Database:** PostgreSQL
- **Security:** Spring Security, JWT (JSON Web Tokens)
- **Testing:** Katalon Studio (Automated Testing)

## ✨ Key Features
- **Secure Authentication:** User login and registration with JWT-based session management and token validation.
- **Transaction Processing:** Securely handle financial operations including deposits, withdrawals, and internal transfers.
- **History Tracking:** High-performance data retrieval for user transaction logs using optimized Postgre schemas.
